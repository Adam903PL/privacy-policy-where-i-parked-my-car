Privacy Policy

Effective date: 19.09.2025

1. Data Collected

The app Where I Parked My Car collects user location data solely for the purpose of:

  • saving selected locations,
  • guiding the user to the saved location using maps.

2. Data Storage

All location data is stored locally on the user's device. The app does not transmit location data to any server or third party.

3. Data Usage

Collected location data is used only for app functionality (saving locations and navigation to them).

4. Data Sharing

The app does not share location data with any third party and does not use any analytics or advertising services that collect user data.

5. User Rights

Users can at any time:

  • delete saved locations from the app,
  • uninstall the app, which will remove all local data.
